# DecodeLabs-Internship_week2
 E-Commerce EDA Project Overview

This project analyzes an E-Commerce dataset to discover customer purchasing patterns, sales trends, and business insights using Exploratory Data Analysis (EDA) and an interactive Streamlit dashboard.
streamlit link:https://decodelabs-internshipweek2-fnbfcnyumysguyqwp9h3hx.streamlit.app/

Objectives

* Identify top-selling products
* Analyze payment method preferences
* Study monthly sales trends
* Evaluate referral source performance
* Detect outliers and correlations in data

 Dataset

* 1,200 transaction records
* 14 features including Product, Quantity, Payment Method, Order Status, Referral Source, and Total Price

 Tools Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn, Plotly
* Streamlit

Key Findings

* Printers were the most purchased products.
* Online payment was the most preferred payment method.
* Instagram generated the highest customer referrals.
* Several high-value transaction outliers were identified.
* Cancelled and returned orders were relatively high and may require further investigation.

Dashboard Features

* KPI Cards
* Revenue Trend Analysis
* Product Performance Analysis
* Payment Method Distribution
* Referral Source Analysis
* Correlation Heatmap
* Interactive Filters

 Conclusion

The project demonstrates how EDA can transform raw transaction data into valuable business insights and support data-driven decision-making through interactive visualizations.
